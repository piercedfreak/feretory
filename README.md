# feretory

feretory is a desktop monitoring tool designed to detect Diablo IV free cosmetic rewards, promotions, event drops, community discoveries, and related signals from public text sources.

## Features

* Blizzard news monitoring
* Diablo IV forum monitoring
* Reddit monitoring
* Relevance scoring engine
* Alert levels (LOW / MEDIUM / HIGH)
* Background scanning
* Configurable scan intervals
* Windows desktop app (.exe)

## Install

### Run from source

```bash
npm install
npm start
```

### Build Windows EXE

```bash
npm run dist
```
